Installation Guides for OCR Tools

Core Dependencies

Poppler (Required for PDF processing)

  • macOS: brew install poppler
  • Ubuntu/Debian: sudo apt-get install poppler-utils
  • Windows: Download from poppler-windows and add to PATH

Tesseract OCR

  • macOS: brew install tesseract
  • Ubuntu/Debian: sudo apt-get install tesseract-ocr
  • Windows: Download installer from UB-Mannheim

Language Data: For non-English languages:

  • macOS: brew install tesseract-lang
  • Ubuntu/Debian: sudo apt-get install tesseract-ocr-[lang]
    Replace [lang] with language code (e.g., fra, deu, spa)
  • Windows: Additional language data files available in the installer

OCRmyPDF

Requires Tesseract and Poppler to be installed first.

  • macOS: brew install ocrmypdf
  • Ubuntu/Debian: sudo apt-get install ocrmypdf
  • Python (all systems): pip install ocrmypdf

EasyOCR

Python-based OCR with multilingual support.

  • All systems: pip install easyocr
  • On first use, EasyOCR will download required language models automatically
  • Requires PyTorch - will be installed as a dependency

PaddleOCR

High-performance OCR system from Baidu.

  • All systems: pip install paddlepaddle paddleocr
  • For GPU support: pip install paddlepaddle-gpu
  • On first use, PaddleOCR will download required models automatically
