Installation Guide

Quick Install

We've created an automatic installer script to set up all dependencies correctly. 

python install_dependencies.py

This will install Python dependencies including pdf2image and pytesseract.

Python Dependencies

Required Python Packages
pip install pdf2image pytesseract Pillow python-docx Flask

If pdf2image is not working, make sure Poppler is installed (see below)

Installing Poppler

macOS
brew install poppler
Ubuntu/Debian
sudo apt-get install poppler-utils
Windows
  1. Download from GitHub
  2. Extract the contents to a directory (e.g., C:\Program Files\poppler)
  3. Add the bin directory to your PATH environment variable

Installing Tesseract OCR

macOS
brew install tesseract

For additional languages:

brew install tesseract-lang
Ubuntu/Debian
sudo apt-get install tesseract-ocr

For additional languages:

sudo apt-get install tesseract-ocr-[lang]

Replace [lang] with language code (e.g., fra for French)

Windows
  1. Download installer from UB-Mannheim
  2. Run the installer and select additional languages if needed
  3. Ensure "Add to PATH" is checked during installation

Troubleshooting

PDF2Image Not Working

If you get errors about pdf2image or Poppler:

  1. Make sure Poppler is installed correctly for your platform
  2. Check that the Poppler binaries are in your system PATH
  3. On Windows, you may need to set the path explicitly in your code
  4. Try running the test script: python ocr_test.py
