Supported by multiple OCR engines - Converts PDF documents to editable formats.

Core requirements: poppler (for PDF conversion), tesseract (for OCR), and other optional OCR engines.

Installation Guides for OCR Tools

Core Dependencies

Poppler (Required for PDF processing)

macOS: brew install poppler
Ubuntu/Debian: sudo apt-get install poppler-utils
Windows: Download from poppler-windows and add to PATH

Tesseract OCR

macOS: brew install tesseract
Ubuntu/Debian: sudo apt-get install tesseract-ocr
Windows: Download installer from UB-Mannheim

Language Data:

macOS: brew install tesseract-lang
Ubuntu/Debian: sudo apt-get install tesseract-ocr-[lang]
Replace [lang] with language code (e.g., fra, deu, spa)
Windows: Additional language data files available in the installer

Kraken OCR

Specialized for historical documents and non-Latin scripts.

All systems: pip install kraken
Additional requirements: May require additional dependencies based on your system. See the official documentation.

Calamari OCR

Deep learning based OCR engine for historical documents.

All systems: pip install calamari_ocr
Note: Requires TensorFlow. Will be installed as a dependency.

PyOCR

A Python wrapper for Tesseract and Cuneiform OCR engines.

All systems: pip install pyocr
Requirements: Requires either Tesseract or Cuneiform to be installed.
